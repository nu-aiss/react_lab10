# Lab 10 - Task 1
## Описание
В этой части лабораторной работы я настроил тестирование React-приложения с использованием Jest и React Testing Library. В качестве примера было создано приложение Todo List с возможностью добавления, удаления и отметки выполненных задач.

## Что сделано
- создан проект на Vite + React + TypeScript
- настроен Jest
- подключен React Testing Library
- создан компонент TodoList
- написаны тесты для основных сценариев работы компонента

## Проверяемые функции
- отображение пустого списка
- отображение начальных задач
- добавление новой задачи через кнопку
- добавление новой задачи через Enter
- запрет добавления пустой задачи
- изменение статуса задачи
- удаление задачи
- обновление счетчика задач
